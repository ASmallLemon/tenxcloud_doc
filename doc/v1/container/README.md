# 容器服务
时速云的容器服务基于目前异常火热的Docker容器技术，结合Docker生态圈的相关工具及自主研发的方案，提供了一整套的容器集群部署和管理方案。目前提供了诸如云主机、云数据库、Web服务器和博客应用的多样化容器服务，时速云平台能以秒级部署并运行这些容器应用，并支持服务的集群部署、负载均衡、灾难恢复和弹性伸缩。
